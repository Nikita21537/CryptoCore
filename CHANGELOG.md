# Changelog

Все заметные изменения в проекте CryptoCore будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Добавлено
- Комплексный набор тестов с покрытием >90%
- Полная документация API в docs/API.md
- Руководство пользователя в docs/USERGUIDE.md
- Руководство разработчика в docs/DEVELOPMENT.md
- Тесты производительности и безопасности памяти
- Интеграционные тесты для CLI
- Тестовые векторы NIST и RFC
- Скрипт запуска всех тестов tests/run_tests.py
- Документы: CONTRIBUTING.md, SECURITY.md

### Изменено
- Рефакторинг структуры тестов
- Улучшена обработка ошибок
- Оптимизирована производительность AES

### Исправлено
- Утечки памяти в обработке больших файлов
- Ошибки валидации входных данных
- Проблемы совместимости с OpenSSL

## [0.2.0] - 2024-01-15 - Sprint 7

### Добавлено
- Реализация PBKDF2-HMAC-SHA256
- Функция деривации ключей из мастер-ключа
- CLI команда `derive` для получения ключей из паролей
- Поддержка чтения паролей из файлов и переменных окружения
- Тестовые векторы RFC 6070 для PBKDF2
- Интеграционные тесты с OpenSSL

### Изменено
- Обновлена структура проекта с директорией `src/kdf/`
- Улучшена документация в README.md

## [0.1.0] - 2023-12-20 - Sprint 1-6

### Добавлено
- Базовая реализация AES-128/192/256
- Режимы шифрования: ECB, CBC, CFB, OFB, CTR, GCM
- Хеш-функции: SHA-256, SHA3-256
- MAC функции: HMAC-SHA256, CMAC-AES
- Криптографически безопасный ГСЧ
- CLI интерфейс с поддержкой:
  - Шифрование/дешифрование файлов
  - Хеширование
  - Генерация и проверка HMAC
- Базовые юнит-тесты

### Исправлено
- Ошибки выравнивания данных в CBC режиме
- Проблемы с большими файлами в GCM режиме


### Добавлено
- Базовая структура проекта
- Настройка сборки с setup.py
- Конфигурация Git и .gitignore
- Начальная документация
